Beginners Crochet Classes

Crochet is a relaxing therapy that can help many people. Below are some examples of the crochet classes we have.

3/14/2017 10 a.m. - 11 a.m. -- Getting Started
This class focuses on what you need to have to begin any project and will learn to make the slip knot and chain stitch.

3/20/2017 12 p.m. - 1 p.m. -- Single Crochet
This class focuses on how to make the single crochet.

3/23/2017 10 a.m. - 11 a.m. -- Double Crochet
This class focuses on how to make the double crochet.

3/26/2017 12 p.m. - 1 p.m. -- Treble Crochet
This class focuses on how to make the treble croceht.

3/30/2017 1 p.m. - 2 p.m. -- Finishing a Project
This class focuses on the crochet techniques to finish a project these include: casting off, binding off, and how to hide ends of stitches.
